Less Wait For Your Plate-SF

A web app that users can use to report wait times for restaurants. I built both the backend and front-end. Users of the app can search for restaurants by name or category, and all info comes from the Yelp API. I also used Mapbox to create a custom map, and when the user clicks on a restaurant in the result list, a marker that has all the restaurant info as well as the last reported wait time appears on the map.
